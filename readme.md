#Logger
**powerful logging library supports udp,file,stdout and different logging levels per file. Supports c,c++,objective-c. written in c**


##Usage

```
$cd $loggerlib_dir/example;
chmod +x example_run.sh;
./logger_run.sh
```

##Validate

```
$cd $loggerlib_dir/test;
chmod +x test_run.sh;
./test_run.sh
```
